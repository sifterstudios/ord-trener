## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/sifterstudios/ord-trener.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. First-time run
   ```js
   yarn run dev
   ```

- The database will be created automatically but currently take multiple hours to populate words with small distances.
